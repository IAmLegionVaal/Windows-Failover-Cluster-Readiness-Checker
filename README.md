# Windows Failover Cluster Readiness Checker

A read-only PowerShell toolkit for Windows cluster readiness review.

## Features

- Failover Clustering module check
- Cluster service context
- Node readiness checklist
- Network and storage review template
- CSV, Markdown, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Windows_Failover_Cluster_Readiness_Checker.ps1
```

## Safety

Diagnostic-only. It reports readiness context for support review.
